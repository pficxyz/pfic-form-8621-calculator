# PFIC Form 8621 Calculator (DIY)

This repository documents a PFIC Form 8621 calculator and a
do-it-yourself (DIY) workflow for PFIC self-filing.

The focus of this project is to make PFIC Form 8621 calculations
accessible and repeatable through a simple, guided process,
without requiring custom spreadsheets or manual recomputation.

---

## Official Calculator

The hosted PFIC Form 8621 calculator is available at:

https://8621calculator.com/

All calculations, validations, and downloadable Excel workpapers
are generated on the above platform.

---

## How the Calculator Is Used

The calculator is designed to be simple and structured.

In most cases, users only need to:

1. **Upload a transaction CSV** with four required columns:
   - Date  
   - Details  
   - Units  
   - Value  

2. **Map each transaction label** in the “Details” column to the
   appropriate PFIC activity category, following on-screen guidance.

3. **Confirm basic filing inputs**, such as:
   - Reporting tax year  
   - Fund currency  
   - PFIC start year (when PFIC rules first apply)  
   - Ending balance, if applicable  

4. **Run the calculation** and download substantiation-ready
   Excel outputs for Form 8621 reporting.

The interface guides each step. No formulas or manual adjustments
are required from the user.
